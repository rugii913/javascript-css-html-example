media query

media query는 화면 크기(정확하게는 viewport의 size)에 따른 각각의 속성 값을 지정하여, 여러가지 화면을 구성(→ layout과 연관)하는 기술

단순한 형태의 media query syntax - @media [media query modifier] [media type] and ([media condition]) + { [실행문] }

  • @media - media query가 시작됨을 표시
  • [media query modifier] - (생략 가능) 기본 only
  • [media type] - (생략 가능) 기본 all - media query를 해석할 대상
    • all - 모든 미디어 유형에서 사용할 CSS 정의
    • print - 인쇄 장치에서 사용할 CSS 정의
    • screen - print에 해당하지 않는 모든 장치에서 사용할 CSS 정의
    • tty, tv, projection, handheld, braille, embossed, aural, speech는 deprecated
  • and - (생략 가능) 앞과 뒤의 조건을 이어줌
  • ([media condition]) - 해당 조건 설정
  • { [실행문] } - 조건에 따라 실행되는 속성 설정
